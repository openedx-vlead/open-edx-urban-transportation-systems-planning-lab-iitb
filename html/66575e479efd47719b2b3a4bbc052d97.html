Step 1

 Click on ‘Click Here To Perform Experiment with GUI’.
In order to start with the experiments make sure that the input files are available in the format given in the default file. The default file can be downloaded from the table as shown in the image



Step 2

Go to Methods to select the type of experiment to be performed. There are three types of Growth factor method experiment

  • Uniform Growth Factor Method
  • Singly constrained Growth Factor Method
    1. Origin Constrained
    2. Destination Constrained
  • Fratar Growth Factor Method



Step 3

Give the input file after selecting the experiment.

Browse the input file from the required folder and enter the growth factor. Select ‘OK’ after all the input is given
